File nguồn là một bản export CSS/Figma rất dài, gồm palette màu, typography, gradient và effect style. Trang này gom chúng lại thành một cấu trúc tài liệu rõ ràng: token nền tảng ở trên, semantic state ở giữa, và component preview ở cuối để dev hoặc design có thể đọc nhanh và áp dụng trực tiếp.

Semantic states

Các cặp màu sáng/tối cho success, error, warning và disabled phù hợp để dựng alert, input state, badge hoặc status panel.

Success
Light: #05A158
Dark: #15271A
Use: confirm, completed, healthy state
Error
Light: #FF274E
Dark: #220C0C
Use: validation, failed action, critical alert

Light: #FFA602
Dark: #2F240B
Use: caution, pending risk, notice
Disabled
Disabled: #888888
Support gray: #7C828B
Use: inactive buttons, muted unavailable state

Gradient styles

File nguồn có một nhóm color style/gradient riêng. Tôi tách các gradient có tính ứng dụng rõ nhất để nhìn trực quan và code ngay.
